
# MindGuard: AI-Powered Health Companion with Blockchain

**Project Overview**

MindGuard is a solo project developed by Lee Henriques(Me) for the 24-hour online hackathon: **Error 404 Hackfest**. This web application serves as a mental health companion, integrating AI to facilitate mental health check-ins and provide nutrition and fitness tracking. The innovative use of blockchain technology ensures user privacy and data security, making it a unique platform for mental well-being.

---

## Features

- **AI-Powered Health Check-ins**: Users can interact with an AI chatbot to discuss their feelings and receive instant feedback and suggestions.
  
- **Nutrition and Fitness Tracking**: Users can log their food intake and physical activities, receiving AI-driven insights to enhance their health journey.

- **Personal and Global Blockchain**: Data related to user activities is securely stored in a blockchain, with personal blocks viewable only to the user, and a global view showcasing all activities.

- **Dynamic User Interface**: The application features a responsive design, ensuring an optimal user experience across devices, including mobile, tablets, and laptops.

- **Community Engagement**: Users can create posts visible to all, comment on each otherâ€™s posts, and engage in meaningful discussions while maintaining a supportive community atmosphere.

---

## Technology Stack

- **Frontend**: Streamlit
- **Backend**: SQLite, Python
- **AI Integration**: Mental health AI for user interaction
- **Blockchain**: Custom blockchain implementation for secure data storage

---

## Getting Started

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/HenriquesLee/MindGuard-AI-Powered-Health-Companion-with-Blockchain.git
   cd mindguard
   ```

2. **Install Requirements**:
   Ensure you have Python installed and then install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Application**:
   Start the Streamlit app:
   ```bash
   streamlit run app.py
   ```
